## What is ApprenticeVR
ApprenticeVR: VRSrc Edition is designed for managing and sideloading content onto Meta Quest devices

## Installing ApprenticeVR

1.  Run and Install '''ApprenticeVR Setup''' 


2.  Once installed you need to update the '''serverinfo.json''' file 


3. Go to '''C:/apprenticeVrSrc-main'''


4. Copy and replace the "serverinfo.json" with the new one
